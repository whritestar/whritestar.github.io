俞军：产品方法论

第一章：什么是产品经理

  1. 历史上的产品经理
    • 消费时代PM
      • 市场驱动型产品经理
    • 软件时代的PM
      • 项目管理和推进
    • 核心：优化分工，提高效率
  2. 互联网产品经理
    • 互联网时代特性
    • 信息复制分发边际成本低
    • 用户量大
    • 快速迭代
    • 数据和AB测试
  3. 产品经理做什么
    • 产品经理4大职能
      - 需求、生产、销售、协调
    • 用户模型
    • 交易模型

好产品的3个属性：有效用，有收益，可持续性

第2章：企业、产品、用户

  1. 企业、用户和产品之间的关系
    • 企业创造价值产品
    • 企业和用户进行价值交换
  2. 如何理解用户
    • 用户定义
      • 用户是需求的集合
      • 生物人、社会人、认知人、情景人
    • 用户的5个属性
      • 异质性
        • 特点千差万别
      • 情景性
        • 场景不同选择不同
      • 可塑性
        • 用户是可变的
      • 自利性
        • 总效用最大化
      • 有限理性
        • 大脑思考的快与慢
    • 用户行为
      • 用户偏好+认知+情景产生主观期望效用
        • 情景5步
      • 用户分布模型，修正用户模型
    • 用户价值
      • 用户：价值由主观效用决定，也就是欲望的满足程度
      • 主观效用
        • 认知依存
        • 情景依存
        • 经验反馈演化
      • 用户价值=新体验-旧体验-替换成本
  3. 如何理解产品
    • 产品是价值交换的媒介
    • 卖产品是在约束条件下的效用组合
    • 好产品的3个属性
      • 有效用
      • 有利润
      • 可持续
  4. 如何理解企业
    • 发现市场获利机会
      • 洞察
      • 试错
      • 偶然性
    • 生产效率高于市场
      • 权威
      • 显性知识
      • 默会知识
    • 组织效率
      • 共同目标、理念、知识和机制
    • 生存和发展
      • 做了什么
    • 做产品的产出
    • 创造价值
      • 劳动、分工、交易、新技术和制度
  5. 如何用产品进行价值交换
    • 交易的价值是主观的
    • 提升用户主观判断水平，提升用户的理解能力
    • 用户角度：价值交换：效用-成本＞0
    • 企业角度：收益-成本＞0

用户价值=新体验-旧体验-替代成本=效用-成本

第3章：交易

  1. 交易和交易模型
    • 什么是交易
      • 产品即交易
      • 为何需要交易：创造价值，挣钱
    • 交易模型
      • 如何创造利益
      • 如何分配利益
        • 损失厌恶
        • 增加福利
        • 激励相容
    • PM为什么要关注交易
      • 创造用户价值
      • 降低生产和交易成本
      • 企业：影响用户交易行为的ROI排序
  2. 效用
    • 什么是效应
      • 欲望的满足程度
    • 效用多样化：货币，时间，身体需要，信念，认知等
    • 产品效用和用户欲望的不对等性
    • 产品是约束条件下的效用组合
  3. 边际
    • 边际效用和边际效用递减
    • 边际成本
    • 边际利润
  4. 成本
    • 机会成本
    • 交易成本
      • 市场性交易成本
        • 搜索成本和度量成本
        • 寻价成本和决策成本
        • 实施成本和保障成本
      • 管理型交易成本
      • 政治性交易成本
    • 产品经理要关注交易成本
      • 标准化
      • 智能手机
      • 线上化
  5. 供需定律
    • 约束条件下
    • 价格下降，需求上升
  6. 相对价格
    • 相对价格=（直接成本+交易成本）/效用组合
    • 增加/改变效应组合、降低交易成本/相对价格

企业的核心是创造交易，有交易才有用户
