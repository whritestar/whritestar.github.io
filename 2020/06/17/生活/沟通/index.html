沟通难么？复杂么？

KWL
-know
-what
-learn
反省：事前思考过么？有什么疑问么？

为什么沟通会怎么难？为什么这么难理？

  • 因素太多（跨类很多）
  • 乱（又有缠绕）
  • 且有些因素很难
  • 还有对应（人、情景）
  • 表象回归到本质，进行提炼

本质影响沟通的因素：

  1. 沟通的内容本身 

    / 意义的构建（包含问题的挖掘）
/ 解决方案、解决步骤（如果没有意义的构建，对方动力可能就不够强，意愿没有解决掉）

  2. 关系与情感
    每次沟通，以上2个是都有的，“焦不离孟,孟不离焦”
    从沟通的目的也可以看出，两者都是非常重要的

    沟通的目的：

    1. 传递信息；
    2. 取得认同（职务、身份）
    3. 建立改善人际
    4. 达到一种特定的目标
      e.g.马云往下的沟通，是很方便。Why？关系和情感层的确定的，且是促进沟通的。

影响沟通的核心因素，具体是怎么影响沟通的？

一、关系与情感是怎么样影响沟通的？

（一）关系与情感的核心影响因子

  1. 职务
  2. 社会地位
    • -财富
    • -权利
    • -声望
  • e.g. 水平沟通时，社会地位是对等的；

  1. 利益关系（e.g.产品部门与经营部门经常要打架，因为有利益冲突）

  2. 人际关系

  3. 性格不同

    • –害羞，对于突破沟通的初始屏障，难度是较大的

    • –敏感，情绪稳定性（说者无意，听者有心）

    • –主动，对于沟通促进很大

    • –袒露，内心比较深入的东西平时不太敢讲的先讲出来，对于突破沟通初始屏障是很有帮助的

    • –个人好恶

    • –当时的身心状态

    • –沟通过程中引发的情绪应激

    • –是否能够感知、理解、管理‘关系与情绪’（包括 自我的、他人的）

      --管理不一定需要在理解基础上，只要感知到作出反应即可，这叫转移；
--但是最好的是要在理解之上，因为有些是需要搁置冷静的；

内省是人体八大自律之一

二、沟通内容本身是怎么样的？

  1. 表达
    • -意义的构建
      • –第一是意识问题；
      • –第二是思维方向思维模式的问题（问题的发掘与问题的表征、意愿的激发）
        • –方向要正确
        • –价值问题，让人家知道价值本身大小，以及价值跟双方的关系
    • -解决方案、解决步骤
    • -用很好的形式（语言+非语言）来表达，需要吻合‘易懂、
      • –重难点突出
      • –急迫点突出
      • –形成结构化，结构要有序，有先后，主线清晰
      • –要有逻辑（有论点+论据）
        • e/g/ 经营中客户反馈很好但是不行动（1.与我无关；2.很好但不相信）
      • –没有多余的话干扰
      • -一定要吻合沟通对象的认知、观念、知识结构的基础、情感、立场
  2. 理解
    • -理解能力（每个人都不同，稳定而不可控）e.g.不能跟小孩讲高等数学
    • -认知负荷：
      • –内在认知负荷：针对这个问题的知识、经验的积累储备；顺应
      • –外在认知负荷：干扰（有认知负荷的就只能单线层；需要有注意力资源不能有万一的也只能单线层）
    • -认知偏差：
      • –立场、观念、价值判断不同，认知是相反的；
      • –刻板印象，有针对群体、个人的刻板印象；在理解的时候不能够有刻板印象，先入为主，少假设；
      • –文化背景

那如何能够沟通好？

一、准备工作是否充分

  1. 是否清楚沟通的目的
  2. 沟通对象是谁？
    • –关系与情感
    • –沟通内容（理解层面：认知等）
  3. 沟通什么？
    • –沟通意义，需要具像化、形象生动
    • –解决方案、解决步骤
    • –沟通情感情绪（如何让对方有很好的应激情绪：肢体语言等）
  4. 如何沟通？
    • –意义的构建，结构完整性
    • –解决方案、解决步骤
    • –表达方式
  5. 要与对方协商好，沟通渠道（面对面、远程等）、沟通时机（有没有时间、身心状态等）、沟通场所
  6. 事前告知（通常情况下，让人家心理、认知上有准备，是不一样的）

那是不是都要做好呢？只要做的比人家更好，那沟通就会比人家更好

二、沟通过程

  1. 气氛的营造与调节
    • –打破起始的沟通屏障
    • –激发沟通意愿
    • –消除防卫意识
      • –带着抵触的态度来的
    • –避免过程中出现不良的情绪应激
      • –对事不对人
      • –核心‘关系与情绪的感知、理解、管理’
    • –幽默

How？打破起始的沟通屏障、激发沟通意愿？

1. 主动大胆，敢开口
2. 主动袒露
3. 寻找双方都有的大的价值点
4. 实在不行就寻找对对方的价值点
5. 寻找对方的喜好点

How？消除防卫心理，避免过程中出现不良的情绪应激？

1. 平等、信任、专注、坦诚
2. 针对羞涩、敏感的人：热情、友善、亲密 亲密距离（0.4m内）、个人距离（0.4-1.2m）、交际距离（1.2m以上）
3. 对于往下的人：接纳、认同、支持
--认同行为（表达、倾听、反馈）、认同价值、认同能力
4. 对职务高的人：钦佩、尊重、依赖（最终如果形成依赖，失去了批判性思维，那也不好）
  1. 表达
    • –语言表达：口语、书面语
      • –同理心（理解对方的立场、价值观、认知、表达习惯；共情能力，能够感同身受）
      • –语气、语调（抑扬顿挫 VS 白开水；该强调的地方要有变化）
      • –重复的力量（重点难点、紧急点；人家需要深入知道的地方）
      • –集中的力量（10min，8min花在哪里，2min花在哪里；激发意愿时，集中在哪里；
    • –非语言表达：肢体语言、表情、眼神、人际距离
      • –表情，表情也是一种给对方是否理解的回馈
      • –眼神，一直盯着看会看毛，一直没有眼神交流也不行
  2. 倾听与反馈（基于表达方）
    • –态度、情绪的表明与反馈
      • 通过非语言体系和语言体系表明对对方的态度，气氛营造里面的东西（平等、信任…）
    • –理解的表明与反馈
      • 给积极的反馈
      • 少假设、少插嘴、少打断、少评价，先听完整（段落的完整）
      • 多复述、多理解、多寻问（是否是人家的本意）
        • 「插」沟通中的两大原则：
          1. 忠始原则（站在事情的本意上，而不是过程中的表达）
          2. 宽容原则（在搞不拎清的情况下，把利益归于对方）
  3. 推进、推动
    • –引导
      • 主要是在平行沟通中，对于偏害羞的人，要多鼓励，多提问
    • –在理解的基础上+价值反馈+后续行动
      • 如果已经理解了，一定要加上对价值的反馈，以及接下来的行动反馈（这对对方的支持非常厉害）
    • –补充、改进、扩展
      • 补充，是同类要素的补充，主要是流畅性
      • 插-创造性思维：流畅性、灵活性、创新性 e.g. 砖头有什么用？
    • –建设性的提议
    • –不同意见
      • -态度一定要好
      • -原则性的一定要坚持，需要有理有据
      • -有节（当对方情绪比较激动抵触时，先平复克制下再说；尽量避免全盘否定；对事不对人，一定要反复强调，但即使强调对方可能也会有情绪）
      • -非原则性的意见，可以求同存异，表达尽量用描述性语言而不是评价性语言
    • –评价（不一定在补充、改进、扩展等）
      • -评价总体是有点上对下的，评价的好价值是很大的，但是评价不好很容易从事转到人上
      • -那在应对时，采用跟‘不同意见’一样的做好就好
  4. 突发事件

    如何正确地应对突发事件？

    • –首先要以‘沟通’为中心，其他干扰事情能不搞先不搞；非搞不可，先表达歉意，表示对这个事情的重视

三、沟通结束后的总结提升

四、沟通是必需的，但沟通不是万能的（不要以为啥都可以通过沟通解决）

有些事情是需要时间的，不是当下的沟通就能解决问题的
有些东西，拿事实说话，拿行动说话就可以了

应用
首先思考这事儿你能不能沟通？换个人能不能沟通？
一次沟通肯定不可能的

针对中学生这个群体，沟通涉及的对象、场景：

一、沟通对象

  1. 同学
  2. 伙伴（玩伴、志同道合的伴）
  3. 老师
  4. 父母
  5. 亲戚

二、沟通场景

分类，要：尽量独立、完整、适合解决问题

  1. 参加活动（团体or组织）
  2. 活动的‘建设、管理、冲突的处理’
    多重身份
    对职务的认同度
  3. 取得认同、支持
    尤其对于‘自我效能感特别差’、‘失败回避型’的人
  4. 人际的建立、维系、改善、终止
  5. 小矛盾、小摩擦等矛盾冲突的处理；
  6. 家庭关系（与父母、兄弟姐妹等
  7. 求助
  8. 学习知识上的传递与请教

有些偏向人际的沟通，有些偏向内容（工具性）的沟通

跟学生讲，需要有具体的对象、场景、案例，在这之下，去讲背后的原理，如何应用

情绪对抗理论：

  1. 玩过刺激性项目后下来的一身轻松愉悦感
  2. 情绪后的体验会越来越强
    e.g. 吸毒，刚开始是因为吸的过程很爽，但到后面吸过程中的爽是越来越弱，但是不吸时的难受越来越强。是难受逼人去吸，而不是爽。

创造性思维是怎么来的？

  1. 问题的表征
  2. 信息收集（无脑）
  3. 酝酿
  4. 顿悟
